# 🎂🍵 Happy Birthday Hạnh — Bánh Sinh Nhật Matcha 3D

Một chiếc bánh sinh nhật matcha 3 tầng xanh–trắng lung linh dưới trăng, làm bằng cả trái tim để dành tặng người thương.

**🔗 Xem trực tiếp:** https://khai293.github.io/hpbd-em-yeu/

## ✨ Có gì trong món quà này

- 🍵 Bánh matcha 3 tầng xanh–trắng: kem chảy lấm tấm bột matcha, kem tươi xoáy xen kẽ hai vị, dâu tây, việt quất, cherry, macaron, cốm đường, nhũ vàng ăn được, chuỗi trân châu vàng — dựng hoàn toàn bằng Three.js
- 🕯 Nến lung linh có ánh sáng thật — **chạm vào ngọn nến để thổi**; thổi hết sẽ có pháo giấy, lời chúc, và topper trái tim vàng **phun pháo bông** ✨
- 💫 **Sao băng** thỉnh thoảng lướt qua bầu trời — và chắc chắn xuất hiện đúng lúc thổi nến để kịp ước
- 🌕 Trăng tròn tỏa quầng sáng ấm, đom đóm lập lòe quanh bánh, dây đèn vàng nhấp nháy quanh mâm
- ❤ Trái tim hồng & **xanh dương** bay lơ lửng, cánh hoa rơi, hạt bokeh lấp lánh, hiệu ứng bloom
- 🎵 Nhạc Happy Birthday phong cách hộp nhạc có bè trầm (WebAudio, tự cất lên khi chạm màn hình)
- 💌 Tấm thiệp tình yêu với hiệu ứng gõ chữ, biển sô-cô-la trắng khắc tên **Hạnh**
- 📱 Xoay bánh bằng chuột / chạm, chạy mượt trên điện thoại

## 🎁 Tùy biến món quà

**Cách 1 — qua link (không cần sửa code):**

```
https://khai293.github.io/hpbd-em-yeu/?ten=TênKhác&nen=7
```

- `ten` — tên hiển thị trên tiêu đề, biển sô-cô-la, lời chúc và thiệp (mặc định: Hạnh)
- `nen` — số ngọn nến (1–12)

**Cách 2 — sửa trực tiếp:** mở `index.html`, tìm khối `CHỈNH SỬA MÓN QUÀ CỦA BẠN Ở ĐÂY 💝` ngay đầu file để đổi tên, số nến và toàn bộ lời trong thiệp.

## 🛠 Công nghệ

- [Three.js](https://threejs.org/) r160 — dựng cảnh 3D, đổ bóng, bloom, hệ hạt (pháo giấy, pháo bông, sao băng, khói)
- WebAudio API — nhạc hộp nhạc 2 bè tự tổng hợp, không dùng file nhạc
- Một file `index.html` duy nhất, không cần build
