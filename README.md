# 🎂 Happy Birthday Em Yêu — Bánh Sinh Nhật 3D

Một chiếc bánh sinh nhật 3 tầng 3D lung linh, làm bằng cả trái tim để dành tặng người thương.

**🔗 Xem trực tiếp:** https://khai293.github.io/hpbd-em-yeu/

## ✨ Có gì trong món quà này

- 🎂 Bánh kem 3 tầng với kem chảy, kem tươi xoáy, dâu tây, cherry, macaron, cốm đường, chuỗi trân châu vàng — dựng hoàn toàn bằng Three.js
- 🕯 Nến lung linh có ánh sáng thật — **chạm vào ngọn nến để thổi**, thổi hết sẽ có pháo giấy + lời chúc
- 🎵 Nhạc Happy Birthday phong cách hộp nhạc (WebAudio, tự cất lên khi chạm màn hình)
- 💌 Tấm thiệp tình yêu với hiệu ứng gõ chữ
- ❤ Trái tim bay, cánh hoa hồng rơi, hạt bokeh lấp lánh, hiệu ứng bloom
- 📱 Xoay bánh bằng chuột / chạm, chạy mượt trên điện thoại

## 🎁 Tùy biến món quà

**Cách 1 — qua link (không cần sửa code):**

```
https://khai293.github.io/hpbd-em-yeu/?ten=TênNgườiẤy&nen=7
```

- `ten` — tên hiển thị trên tiêu đề, lời chúc và thiệp
- `nen` — số ngọn nến (1–12)

**Cách 2 — sửa trực tiếp:** mở `index.html`, tìm khối `CHỈNH SỬA MÓN QUÀ CỦA BẠN Ở ĐÂY 💝` ngay đầu file để đổi tên, số nến và toàn bộ lời trong thiệp.

## 🛠 Công nghệ

- [Three.js](https://threejs.org/) r160 — dựng cảnh 3D, đổ bóng, bloom
- WebAudio API — nhạc hộp nhạc tự tổng hợp, không dùng file nhạc
- Một file `index.html` duy nhất, không cần build
